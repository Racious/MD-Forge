# Changelog

## v0.3.3

- 啟動時自動還原上次工作階段，重啟不遺失任何分頁與內容
- 關閉含未儲存內容的分頁改用自訂確認對話框
- 分頁列空白處雙擊可快速新增分頁
- 安裝版啟動後靜默檢查更新，有新版詢問是否立即安裝
- 攜帶版 fallback 至引導下載頁，兩者不重複通知
- 修正視窗最小化時喚起行為
- 完善 CI release 流程，自動產生版本說明與上傳攜帶版

## v0.3.0

- 支援 Markdown 中嵌入 HTML（`<p>`、`<img>`、`<details>` 等）
- 新增 DOMPurify 過濾層，移除危險標籤與事件屬性，確保安全渲染
- 新增 GitHub Pages 展示頁（[racious.github.io/MD-Forge](https://racious.github.io/MD-Forge/)）
- 編輯器支援縮放：`Ctrl + 滾輪` / `Ctrl + +` / `Ctrl + -`，`Ctrl + 0` 重設
- TOC 點擊標題時編輯器同步平滑捲動至對應位置
- 首頁 ICON 更換為應用程式圖示
- 修正應用程式圖示在暗色背景下的透明角問題
- 分頁列右側空白處雙擊可快速新增分頁
- 啟動時自動還原上次工作階段（所有分頁與內容）
- 關閉含未儲存內容的分頁時顯示確認對話框
- 安裝版支援應用程式內自動更新

## v0.2.1

- 修正視窗啟動時短暫出現於左上角後才移至中央的問題
- 加入視窗狀態記憶，重新開啟時還原上次關閉前的位置與大小
- 若上次關閉時為最大化，重開後直接以最大化顯示

## v0.2.0

- 新增拖曳開檔（`.md` / `.markdown` / `.mdx`）
- 新增單一實例模式，雙擊第二個檔案開新分頁而非新視窗
- 修正本機圖片（相對路徑）無法顯示的問題
- 修正 Data URI Base64 圖片因 URL 編碼損壞的問題
- Mermaid 安全層級調整為 `strict`

## v0.1.0

- 多分頁編輯、即時預覽、Mermaid 圖表
- 版本資訊顯示（工具列 + 首頁）
- 雙擊開檔（`.md` 檔案關聯）
- 側邊欄捲動改善
