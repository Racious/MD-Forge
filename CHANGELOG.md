# Changelog

## v0.3.9

- 分頁列支援滑鼠拖曳重排：按住分頁左右拖動即可調整順序，拖曳時分頁跟手浮起、鄰近分頁自動讓位空出落點，放開即定位；拖曳中按 `Esc` 可取消還原。新順序會被記住

## v0.3.8

- JSON 格式化：工具列 `Format` 按鈕、`Shift + Alt + F`、命令面板，以及存檔時自動格式化（設定可開關，預設開啟）；JSON 不合法時安全略過、不損內容
- JSON 樹狀檢視支援快速複製：每個節點可複製其 JSON／值／路徑，並以 toast 即時提示「已複製」
- 分頁列新增「+」新分頁按鈕，不論分頁數量皆可開新檔（雙擊空白處仍保留為輔助）

## v0.3.7

- 設定改為右側滑出式抽屜：點擊面板外、`Esc` 或再按一次 `Settings` 皆可關閉，後方內容保持可見
- 新增命令面板（`Ctrl + K`，亦相容 `Ctrl + Shift + P`）：輸入即篩選並執行各項操作，方向鍵選取、`Enter` 執行；關閉分頁仍保留未儲存確認
- 抑制 WebView 預設列印捷徑（`Ctrl + P` / `Ctrl + Shift + P`）

## v0.3.6

- 修正分頁列滑鼠滾輪無法左右捲動的問題
- 設定按鈕改為切換式：再按一次、`Ctrl + ,` 或 `Esc` 皆可關閉，開啟時按鈕高亮
- 新增分頁快捷操作：`Ctrl + W` 關閉當前分頁、`Ctrl + Tab` / `Ctrl + Shift + Tab` 切換分頁、滑鼠中鍵點分頁即關閉

## v0.3.5

- JSON 文件新增樹狀檢視，可逐層展開／縮合，並支援一鍵全部展開／縮合
- JSON 文件支援 Edit / Split / Preview 模式，編輯時樹狀檢視即時同步，解析錯誤即時提示
- 分頁列支援滑鼠滾輪橫向捲動，並新增左右切換按鈕（分頁溢出時顯示）
- 切換分頁時自動將作用中分頁捲入可視範圍，分頁滑過時顯示完整路徑
- 開啟對話框過濾器細分為 All supported / Markdown / JSON / All files
- 存檔對話框依當前文件類型自動預設對應副檔名（編輯 JSON 時預設 `.json`）

## v0.3.3

- 啟動時自動還原上次工作階段，重啟不遺失任何分頁與內容
- 關閉含未儲存內容的分頁改用自訂確認對話框
- 分頁列空白處雙擊可快速新增分頁
- 安裝版啟動後靜默檢查更新，有新版詢問是否立即安裝
- 攜帶版 fallback 至引導下載頁，兩者不重複通知
- 修正視窗最小化時喚起行為
- 完善 CI release 流程，自動產生版本說明與上傳攜帶版

## v0.3.0

- 支援 Markdown 中嵌入 HTML（`<p>`、`<img>`、`<details>` 等）
- 新增 DOMPurify 過濾層，移除危險標籤與事件屬性，確保安全渲染
- 新增 GitHub Pages 展示頁（[racious.github.io/MD-Forge](https://racious.github.io/MD-Forge/)）
- 編輯器支援縮放：`Ctrl + 滾輪` / `Ctrl + +` / `Ctrl + -`，`Ctrl + 0` 重設
- TOC 點擊標題時編輯器同步平滑捲動至對應位置
- 首頁 ICON 更換為應用程式圖示
- 修正應用程式圖示在暗色背景下的透明角問題
- 分頁列右側空白處雙擊可快速新增分頁
- 啟動時自動還原上次工作階段（所有分頁與內容）
- 關閉含未儲存內容的分頁時顯示確認對話框
- 安裝版支援應用程式內自動更新

## v0.2.1

- 修正視窗啟動時短暫出現於左上角後才移至中央的問題
- 加入視窗狀態記憶，重新開啟時還原上次關閉前的位置與大小
- 若上次關閉時為最大化，重開後直接以最大化顯示

## v0.2.0

- 新增拖曳開檔（`.md` / `.markdown` / `.mdx`）
- 新增單一實例模式，雙擊第二個檔案開新分頁而非新視窗
- 修正本機圖片（相對路徑）無法顯示的問題
- 修正 Data URI Base64 圖片因 URL 編碼損壞的問題
- Mermaid 安全層級調整為 `strict`

## v0.1.0

- 多分頁編輯、即時預覽、Mermaid 圖表
- 版本資訊顯示（工具列 + 首頁）
- 雙擊開檔（`.md` 檔案關聯）
- 側邊欄捲動改善
