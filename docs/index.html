MD Forge 是一款以本機檔案為核心的桌面 Markdown 編輯器。 支援即時預覽、多分頁、Mermaid 圖表與 HTML 匯出，讓寫作、整理與技術文件都更安靜而流暢。 MD Forge is a local-first desktop Markdown editor with live preview, multi-tab editing, Mermaid diagrams, and HTML export — built for quiet, focused writing.

功能特色 Features

為技術文件與日常筆記打造。 Built for docs, notes, and focused writing.

寫作需要的一切，保留清楚的流程；不必要的干擾，則保持在視線之外。 Everything you need for Markdown writing, without adding noise to your workflow.

01

多分頁編輯Multi-tab Editing

同時開啟多個文件，相同檔案不重複開啟。關閉所有分頁後自動回到首頁。Open multiple documents at once, avoid duplicate tabs, and return home when every tab is closed.

02

即時預覽Live Preview

Edit / Split / Preview 三種模式隨時切換，Split 模式下左右同步捲動。Switch between Edit, Split, and Preview. Split mode keeps editor and preview in sync.

03

Mermaid 圖表Mermaid Diagrams

流程圖、循序圖、甘特圖等直接在 Markdown 中繪製，亮暗主題自動切換。Render flowcharts, sequence diagrams, Gantt charts, and more directly in Markdown.

04

完整圖片支援Full Image Support

本機圖片、網路圖片、Data URI Base64 圖片均可正確顯示。Local images, remote images, and Data URI Base64 images all render correctly.

05

TOC 側邊欄TOC Sidebar

自動從標題生成目錄，點擊跳至對應段落，支援中文等 Unicode 錨點。Auto-generate a table of contents with Unicode anchor support.

06

彈性開檔Flexible File Opening

雙擊關聯、拖曳、命令列傳入路徑，皆可在同一視窗新分頁開啟。Open by file association, drag and drop, or CLI path — all in one window.

07

主題與設定Themes & Settings

亮色 / 暗色主題、字體大小、自動換行與使用偏好皆持久化儲存。Light/dark themes, font size, word wrap, and preferences are persisted locally.

08

HTML 匯出HTML Export

一鍵匯出為含完整樣式的獨立 HTML 檔案，直接在瀏覽器開啟分享。Export a styled, standalone HTML file for opening and sharing in any browser.

使用流程 Workflow

打開、撰寫、預覽、輸出。 Open. Write. Preview. Export.

📂 開啟本機文件Open local files

透過拖曳、雙擊或命令列快速載入 Markdown 文件。Load Markdown files by drag-and-drop, double-click, or command line.

✍️ 專注編輯與預覽Edit with preview

在 Split View 中同步編輯與預覽，技術文件與流程圖都能即時確認。Write in split view and validate technical docs or diagrams instantly.

📤 匯出 HTML 分享Export and share

將文件輸出為可獨立開啟的 HTML，方便歸檔、交付或分享。Export to standalone HTML for archiving, delivery, or sharing.
